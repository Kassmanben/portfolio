Wikileek

I built this Twitter bot as a tool to demonstrate Markov chain models. It takes in information from the Wikipedia text dump (a collection of text from every article on Wikipedia) and parses it into a SQL database that tracks the frequency with which the word is preceded and followed by another word. This information is used to generate tweets in the style of Wikipedia articles. The app is written using Node.js and mounted on Heroku.
